## C 语法
   按照C语法指定的格式编写代码。 

#### 词法元素：代码由以下5个部分组成,叫做标记符
    • Identifiers:	标识符，代码中自己定义的符号，比如函数，变量，数字  	
    • Keywords:	 关键字，C 语言语法自己用的符号，比如 int ,float,struct,static,void,sieof
    • Constants: 常量，定义的字符、数字等，不可修改的内容, 'm' , 5,  	
    • Operators: 运算符，+，-，*，/
    • Separators: 分隔符，( ) [ ] { } ; , . :  	
    • White Space: 空格，让代码更容易阅读
